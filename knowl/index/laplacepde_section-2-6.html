We can also detect when the problem is ill-posed in the sense that it has no solution. Let us change the equation to
\begin{equation*} \begin{aligned} & -y_t + y_x = 0, \qquad \text{for } x > 0, \enspace t > 0, \\ & y(0,t) = \sin(t), \qquad y(x,0) = 0 . \end{aligned} \end{equation*}
Then the problem has no solution. First, let us see this in the language of SectionÂ 1.9. The characteristic curves are \(t=-x+C\text{.}\) If \(\tau\) is the the characteristic coordinate, then we find the equation \(y_\tau = 0\) along the curve, meaning a solution is constant along characteristic curves. But these curves intersect both the \(x\)-axis and the \(t\)-axis. For example, the curve \(t=-x+1\) intersects at \((1,0)\) and \((0,1)\text{.}\) The solution is constant along the curve so \(y(1,0)\) should equal \(y(0,1)\text{.}\) But \(y(1,0) = 0\) and \(y(0,1) = \sin(1) \not= 0\text{.}\) See FigureÂ 3.7.
