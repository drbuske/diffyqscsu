Example 3.1.4.

A common function is the unit step function, which is sometimes called the Heaviside function
 2 
The function is named after the English mathematician, engineer, and physicist Oliver Heaviside (1850–1925). Only by coincidence is the function “heavy” on “one side.”
. This function is generally given as
\begin{equation*} u(t) = \begin{cases} 0 & \text{if } \; t < 0 , \\ 1 & \text{if } \; t \geq 0 . \end{cases} \end{equation*}
Let us find the Laplace transform of \(u(t-a)\text{,}\) where \(a \geq 0\) is some constant. That is, the function that is 0 for \(t < a\) and 1 for \(t \geq a\text{.}\)
\begin{equation*} \mathcal{L} \bigl\{ u(t-a) \bigr\} = \int_0^{\infty} e^{-st} u(t-a) \, dt = \int_a^{\infty} e^{-st} \, dt = \left[ \frac{e^{-st}}{-s} \right]_{t=a}^\infty \\ = \frac{e^{-as}}{s} , \end{equation*}
where of course \(s > 0\) (and \(a \geq 0\) as we said before).
