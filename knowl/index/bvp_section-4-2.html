For basic Fourier series theory we will need the following three eigenvalue problems. We will consider more general equations and boundary conditions, but we will postpone this until ChapterÂ 6.
\begin{equation} x'' + \lambda x = 0, \quad x(a) = 0, \quad x(b) = 0 ,\tag{5.1} \end{equation}
\begin{equation} x'' + \lambda x = 0, \quad x'(a) = 0, \quad x'(b) = 0 ,\tag{5.2} \end{equation}
and
\begin{equation} x'' + \lambda x = 0, \quad x(a) = x(b), \quad x'(a) = x'(b) .\tag{5.3} \end{equation}
A number \(\lambda\) is called an eigenvalue of (5.1) (resp. (5.2) or (5.3)) if and only if there exists a nonzero (not identically zero) solution to (5.1) (resp. (5.2) or (5.3)) given that specific \(\lambda\text{.}\) A nonzero solution is called a corresponding eigenfunction.
