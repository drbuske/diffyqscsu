Before we tackle the Fourier series, we study the so-called boundary value problems (or endpoint problems). Consider
\begin{equation*} x'' + \lambda x = 0, \quad x(a) = 0, \quad x(b) = 0, \end{equation*}
for some constant \(\lambda\text{,}\) where \(x(t)\) is defined for \(t\) in the interval \([a,b]\text{.}\) Previously we specified the value of the solution and its derivative at a single point. Now we specify the value of the solution at two different points. As \(x=0\) is a solution, existence of solutions is not a problem. Uniqueness of solutions is another issue.
